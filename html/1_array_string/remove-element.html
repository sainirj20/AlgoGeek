27. Remove Element Link to LeetCode Given an array and a value, remove all instances of that value in place and return the new length. (Note: The order of elements can be changed. It doesn't matter what you leave beyond the new length.) Analysis This problem can be solve by using two indices. public int removeElement(int[] A, int elem) { int i=0; int j=0; while(j < A.length){ if(A[j] != elem){ A[i] = A[j]; i++; } j++; } return i; }
